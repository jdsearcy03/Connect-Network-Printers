Auto-Connect Printers

This tool was developed to assist an IT team with connecting network printers. It checks each User account in the Registry for any connected printers. It filters that list for any duplicates, and formats the list into a .vbs print script. After selecting which printers to keep in the list, the print script is saved in a selected location and you can optionally also save it in the Startup folder of the target Computer so that the printers will be auto-connected for all Users at logon.

Query Computer Name

To run the application, you need to enter the name of a Computer that is online.

If you do not enter a Computer Name or if the Computer is not connected to your network, you will receive an error.

Otherwise, the process will initialize.

(Depending on the number of Users, this process could take several moments)

Check Printer List

Once the process is complete, each connected printer will be added as a separate line item in the Print Script box.

If you see any printers that you don't want in the list, just click on them.

If you click on a printer, you will be prompted to confirm that you want to remove that printer from the list.

Once you are happy with the printer list, click the Confirm Button.

Save Print Script

When you click the Confirm button, the print script is automatically saved in a Files folder under the same directory that the script is in.

(This can easily be changed to any location.)

Finally, you are prompted with the option to save the print script to the startup folder of the target Computer.

(This automates printer connection for any future User logon.)
