Auto-Connect Printers

This tool was developed to assist an IT team with connecting network printers. It checks each User account in the Registry for any connected printers. It filters that list for any duplicates, and formats the list into a .vbs print script. After selecting which printers to keep in the list, the print script is saved in a selected location and you can optionally also save it in the Startup folder of the target Computer so that the printers will be auto-connected for all Users at logon.

Query Computer Name

In order for the application to run, you need to provide three pieces of information: Computer Name(s), App Name(s), and whether you want to Install or Uninstall.

To add a Computer that you want to install an application onto, enter its name into the search bar and click on the down arrow.

This will add the Computer Name to the list in the box below the search bar.

You can add as many Computer Names to this list as you like.

Add Applications

Use the search bar in the top right side of the window to query SCCM for available applications.

Enter an application name, part of a name, or series of characters to query SCCM.

Once you find an application that you want to install, simply select that application from the list to highlight it.

Next, select a Computer from the list you previously created.

Finally, select either "Install" or "Uninstall" to initialize the operation.

Both the selected Computer and App remain highlighted after making your selection so that you can easily install the same application on multiple computers or install multiple applications on the same computer.

Check Installation Status

The bottom window displays a running list of actions that have been performed.

Clicking on an entry will display the current installation status of that application on the specified computer.

Once the installation status changes, the line item also updates to reflect that the process has completed.
